Атрибут color тега font

Атрибут color, тега <font>, задает цвет текста.

Указать значение можно одним из следующих способов:

  • имя цвета (black);
  • 16-ричное значение цвета, перед которым необходимо поставить решетку (#000000).

Можно попробовать использовать такую конструкцию <font color="#BruceWillis"> #BruceWillis </font>, или, например, такую <font color="#CANDY"> #CANDY </font>. В таком случае программа выведет следующее:

#BruceWillis
#CANDY

Если в атрибут тега передано некорректное значение (как в примерах выше), браузер не может понять что это за значение и пытается преобразовать его в шестнадцатеричный код с помощью алгоритма, где пропускаемые цифры распознаются как 0, некорректные символы распознаются как 0.

Подробнейший разбор алгоритма разбора приведен по ссылке. Если применить эти правила, то получим следующее:

  • В шестнадцатеричной системе счисления используются цифры от 0 до 9 и латинские буквы от A до F. Заменяем все неправильные шестнадцатеричные символы нулями 0, BruceWillis превращается в B00ce000000;
  • “Добиваем” число нулями до количества знаков, делимого на 3 (11 -> 12), получаем B00ce0000000
  • Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: (B00c, e000, 0000)
  • Обрезаем каждый из аргументов справа, оставляя 2 символа, и получаем результат: #B0e000.

Для #CANDY делаем аналогично:

  • Все неправильные шестнадцатеричные символы нулями 0 и получаем CA0D0;
  • “Добиваем” число нулями до количества знаков, делимого на 3, получаем CA0D00;
  • Разбиваем на три группы, каждая из которых отвечает за одну компоненту RGB: (CA, 0D, 00);
  • Обрезаем каждый из аргументов справа, оставляя 2 символа, и получаем результат: #CA0D00.

Исходя из выше изложенного, если использовать цвет "#CANDY", "#CAND", "#CA0D00", то цвет будет одинаковым: #CANDY, #CAND, #CA0D00 .

Если использовать цвет "#BruceWi", "#BruceWil", "#BruceWillis", #B0e000, то цвет тоже будет одинаковым: #BruceWi, #BruceWil, #BruceWillis, #B0e000.
