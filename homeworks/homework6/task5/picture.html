Домашнее задание : picture

Описание

HTML-элемент picture служит контейнером для одного или более элементов source и одного элемента img для обеспечения оптимальной версии изображения для различных размеров экрана. Браузер рассмотрит каждый из дочерних элементов source и выберет один, соответствующий лучшему совпадению; если совпадений среди элементов source найдено не будет, то будет выбран файл, указанный атрибутом src элемента img . Затем выбранное изображение отображается в пространстве, занятом элементом img.

Чтобы выбрать оптимальное изображение, user agent анализирует атрибуты srcset, media, и type элемента source и выбирает совместимое изображение, которое наилучшим образом соответствует текущему макету страницы, характеристикам устройства отображения и т. д.

Пример

Рассмотрим работу следующего кода:

picture
source srcset="../img/star1.png" media="(min-width: 1100px)"
source srcset="../img/star0.png" media="(min-width: 1000px)"
source srcset="../img/star1.png" media="(min-width: 900px)"
source srcset="../img/star0.png" media="(min-width: 800px)"
source srcset="../img/star1.png" media="(min-width: 700px)"
img src="../img/star0.png" alt="Звезда"
/picture

Согласно коду, если минимальная ширина рабочей области браузера составит более или 1100px, то будет выведена картинка "star1.png" (закрашенная звезда) из первого тега source и дальше условия проверяться не будут. Если ширина экрана будет в диапазоне 1000-1100px, то выведется картинка "star0.png" (незакрашенная звезда). И так далее по условиям. Но если ни одно из условий не подойдет, т.е. экран будет менее 700px, то выведется картинка указанная в теге img.

Попробуйте изменить размеры окна браузера и посмотрите за поведением картинки.
